CPI Display Software
The CPI display software is written in IDL.  This code can be run without a license, by downloading Exelis' IDL Software.  The IDL software includes the IDL Virtual Machine, which can be run without an IDL license. If you already have an IDL license, you will only need to download the CPI display software. 

  1. Download CPI display software.
  2. Follow these instructions for downloading and installing the IDL software.
  3. Follow these instructions for running the IDL Virtual Machine software on your operating system.
