CPI Data and Software
The CPI display software is written in IDL.  This code can be run without a license, by downloading RSI's IDL Virtual Machine Software.  If you already have an IDL license, you will only need to download the CPI display software and the CPI data files. 

  1. Download CPI display software.
  2. Download IDL Virtual Machine software.
  3. Download CPI data.
