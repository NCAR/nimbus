Software for analysis and display of data from Small Ice Detector - 2 HIAPER (SID2H)
The compressed archive file SID2H_v4.1.0.4.zip contains files for installing the program SID2W on MS-Windows computers.Â  The zip file contains:
setup.exe
SID2H Setup.msi
vcredist_x86/
vcredist_x86/vcredist_x86.exe
