xpms2d Download Page

System Requirements

To build xpms2d from source requires an X-Windows based operating system using the Motif toolkit. Most commercial Unices have Motif installed.

OpenMotif is available for RHEL via yum(1) ("yum install openmotif openmotif-devel"). For Fedora, visit RPM find and download the openmotif and openmotif-devel RPMs.

Change log

  • 12/05/2010, 2.5.5: Change in particle rejection criteria, removed obsolete streaker detection. Added simple stuck bit detection. Fix bug in output of Fast2DC hex, only half record was being displayed. Support for "Save PNG" on 64 bit computers.
  • 02/05/2010, 2.5.4: Modify area-ratio-rejection to use circle fit. x64 bit compatibility.
  • 09/09/2009, 2.5.2: Support for files greater than 2GB. Support for gzipped ADS3 files.
  • 06/15/2009, 2.5.1: Ongoing fixes and improvements for Fast2DC. Added "View Histogram" to view text of counts histogram.
