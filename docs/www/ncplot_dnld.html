ncplot Download Page

System Requirements

To build ncplot from source requires that netCDF 3.0 or later be installed, ncplot is X-Windows based using the Motif toolkit. Most commercial Unices have Motif installed.

OpenMotif is available for RHEL via yum(1) ("yum install openmotif-devel"). For Fedora, visit the PBone repository and retreive the ccrma openmotif RPM.
