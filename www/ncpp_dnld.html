System Requirements

To build ncpp from source requires that netCDF 3.0 or later be installed, ncpp is X-Windows based using the Motif toolkit. Most commercial Unices have Motif installed.

OpenMotif is available for RHEL via yum(1) ("yum install openmotif"). For Fedora, visit the PBone repository and retreive the ccrma openmotif RPM.
