Aeros Downloads for Linux and Windows

Here are downloads for Linux and MS Windows versions of the Aeros software.

Software and databases are also provided for the GMT Generic Mapping Tools. These must be installed if you wish to have support for geo-political boundary displays in the Aeros track plot. If you do not install GMT, Aeros will still work fine, but just not provide this feature.

Fedora Core 3

Linux GMT (65MB): gmt.tar.gz
  • Set $GMTHOME to where you installed the above tar file.

RHEL 5

Linux Aeros: aeros-4082.rhel5.tar.gz

Fedora 8

Linux Aeros: aeros-4082.f8.tar.gz

Microsoft Windows XP

Windows Aeros (9MB): aeros-4480.msi Qt4
Windows Aeros (9MB): aeros-4428.msi Qt4 beta
Windows Aeros (14MB): aeros-4082.msi Last Stable Qt3

Windows GMT (48MB): aeros-GMT.msi
Note that this msi will set the GMTHOME environment variable to the target directory. If you already have GMT installed, and the GMTHOME variable is set, you do not need to install this msi; your existing GMT should work fine.
